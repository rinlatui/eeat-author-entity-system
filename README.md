## What This Repository Solves

Many websites add an author name at the bottom of an article, but they do not build a real author entity system.

This repository helps create a connected system:

```text
Author page
+ Author box
+ Reviewer box
+ Editorial policy
+ About page
+ Contact page
+ Organization entity
+ Person schema
+ ProfilePage schema
+ Article author markup
+ sameAs links
+ GitHub / LinkedIn / public profiles
+ Multi-domain consistency
+ Evidence layer
```

## Quick Start

1. Read `docs/01-eeat-author-entity-overview.md`
2. Create an author profile using `templates/author-profile-template.md`
3. Map sameAs links using `templates/sameas-map-template.csv`
4. Review schema with `checklists/schema-validation-checklist.md`
5. Implement author/reviewer boxes with the files in `wordpress/`
6. Audit the system using `audits/author-entity-audit-report-template.md`
7. Track improvements using `templates/author-entity-dashboard-template.csv`
