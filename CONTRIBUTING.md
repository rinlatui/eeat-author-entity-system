# Contributing

This repository is a public EEAT Author Entity System.

## How to Contribute

You can improve:

- Author entity framework notes
- Reviewer workflow
- WordPress implementation snippets
- Schema templates
- Audit templates
- Prompt examples
- Multi-domain consistency examples

## Content Principles

- Keep examples practical.
- Do not create fake credentials.
- Do not recommend fake authors or fake reviewers.
- Do not include private client data.
- Use schema to support visible content, not to replace it.
- Avoid unsupported ranking guarantees.

## Commit Style

Use clear commit messages:

```text
Add reviewer box snippet
Improve sameAs mapping template
Update author entity audit checklist
```
