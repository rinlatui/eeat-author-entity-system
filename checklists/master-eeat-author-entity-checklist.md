# Master EEAT Author Entity Checklist

## Author Identity

- [ ] Author name is consistent
- [ ] Author bio is specific
- [ ] Author photo is consistent where used
- [ ] Job title is clear
- [ ] Expertise topics are visible
- [ ] Main author profile is defined

## Author Page

- [ ] Author page is accessible
- [ ] Author page is indexable if used for entity/EEAT
- [ ] Author page lists articles
- [ ] Author page links to public profiles
- [ ] Author page explains role and expertise

## Reviewer

- [ ] Reviewer role is defined
- [ ] Reviewer box is visible where needed
- [ ] Review process is documented
- [ ] Reviewer profile is verifiable

## sameAs

- [ ] GitHub profile connected where relevant
- [ ] LinkedIn profile connected where relevant
- [ ] Other author pages connected
- [ ] sameAs links return valid pages
- [ ] sameAs names are consistent

## Schema

- [ ] Person schema reviewed
- [ ] ProfilePage schema reviewed
- [ ] Article author schema reviewed
- [ ] Organization schema reviewed
- [ ] Schema matches visible content
