# Author / Reviewer Workflow Template

## Workflow

```text
Topic selection → Author draft → Editorial review → Technical/source review → Publish → Update → Re-review
```

## Responsibility Matrix

| Step | Owner | Output |
|---|---|---|
| Draft | Author | Article draft |
| Review | Reviewer | Review note |
| Publish | Editor | Published article |
| Update | Author/Editor | Updated content |
